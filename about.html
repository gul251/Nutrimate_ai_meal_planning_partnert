About NutriMate

A student-built wellness platform focused on practical nutrition planning.

1

Project Vision

NutriMate was designed to make meal planning simpler for students and young professionals who need quick, realistic, and affordable nutrition guidance.

2

What Problem We Solve

Many users struggle to stay consistent with healthy eating. NutriMate combines daily planning, goal tracking, and guided suggestions in one simple dashboard.

3

How It Works for Users

Users create a profile, set diet and activity preferences, define calorie/protein goals, and then receive meal suggestions with local fallback support when AI is limited.

4

Technology Stack

NutriMate uses HTML/CSS/JavaScript on the frontend, Firebase for authentication and data storage, and a secure backend proxy pattern for AI provider integration.

5

FYP Focus

This project is built as an FYP with emphasis on reliability, accessibility, and real-world usability. The roadmap includes continuous UI improvements and stronger trust pages.
